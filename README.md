# Ships Detector

Приложение на основе Ultralytics YOLO для нахождения кораблей на спутниковых изображениях

```
 Usage: main.py [OPTIONS] IMAGE SAVE_DIR

 Распознавание кораблей на спутниковых изображениях

╭─ Arguments ──────────────────────────────────────────────────────────────────────────────────────────────────────────╮
│ *    image         TEXT  Путь до изображения, на котором необходимо распознать корабли [required]                    │
│ *    save_dir      TEXT  Путь для сохранения готового изображения [required]                                         │
╰──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────╯
╭─ Options ────────────────────────────────────────────────────────────────────────────────────────────────────────────╮
│ --help          Show this message and exit.                                                                          │
╰──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────╯
```
